Powstawanie

Kąt ostry między poszczególnymi odcinkami ma alfa stopni. Powtarzam fraktal 5 razy, z przesunięciem o 72 stopnie tak, aby powstał płatek śniegu.

Kod w języku python 

def y(a, alfa):
    fd(a)
    lt(alfa)
    fd(a / 2)
    bk(a / 2)
    rt(2 * alfa)
    fd(a / 2)
    bk(a / 2)
    lt(alfa)
    bk(a)
def galazka(n, a, alfa):
    if n == 0:
        fd(a)
        bk(a)
    else:
        lt(alfa)
        y(a / 3, alfa)
        rt(alfa)
        fd(a)
        lt(alfa)
        y(a / 2, alfa)
        rt(alfa)
        bk(a / 2)
        lt(alfa)
        galazka(n - 1, a / 2, alfa)
        rt(2 * alfa)
        galazka(n - 1, a / 2, alfa)
        lt(alfa)
        fd(a / 2)
        rt(alfa)
        y(a / 2, alfa)
        lt(alfa)
        bk(a)
        rt(alfa)
        y(a / 3, alfa)
        lt(alfa)
